# PACE-preTAVR Risk Calculator

Calculadora de riesgo para procedimientos TAVR (Transcatheter Aortic Valve Replacement).

## Descripción

Esta aplicación web permite calcular el riesgo de implantación de marcapasos permanente (PPI) después de un procedimiento TAVR utilizando el modelo PACE-preTAVR.

## Despliegue

Este proyecto está configurado para desplegarse automáticamente en Netlify.

[![Netlify Status](https://api.netlify.com/api/v1/badges/your-badge-id/deploy-status)](https://app.netlify.com/)

## Tecnologías

- HTML5
- CSS3
- JavaScript (Vanilla)

## Uso

Simplemente abre `index.html` en un navegador web o visita el sitio desplegado en Netlify.
